# yEvo Mutation Browser Development README

To add functionality to the yEvoMutation browser, 3 things need to be done:

1. You need to create a file containing the UI and server functions \(see chromMap.R for an example\)
2. You need to add a call to the UI function in the tab panel within the UI section of the main App.R file
3. You need to add a call to the server function in the server section of the main App.R file
